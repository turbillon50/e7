# Logo de marca — E7 Pictures

Sube aquí el logo oficial. El componente `Logo` lo detecta automáticamente.

## Cómo subirlo

1. Coloca el archivo en esta carpeta (`public/brand/`) con uno de estos nombres exactos:
   - **`logo.svg`** — preferido (vectorial, nítido en cualquier tamaño)
   - **`logo.png`** — alternativa (fondo **transparente**)
2. Al hacer commit a `main`, Vercel redespliega y el logo aparece en el header,
   el footer y las pantallas de login/registro.

## Especificaciones recomendadas

- Fondo **transparente**.
- Versión **clara o champagne gold** (la interfaz es oscura, `#131313`).
- Si es PNG: alto mínimo ~120 px (idealmente 240+).
- Proporción horizontal o cuadrada (se escala por altura, ~30 px en el header).

Mientras no exista el archivo, la app muestra el wordmark tipográfico
"E7 PICTURES" como respaldo, así que nunca se ve una imagen rota.
